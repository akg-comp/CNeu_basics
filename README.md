# CNeu_basics
Computational Neuroscience Notes
Basic neurodynamic models - documentation and codes


Models implemented:
- Leaky Integrate-and-Fire neurons
- Spike-frequency adaptation
- Synaptic conductance models
- Hopf bifurcations in firing-rate systems
- Hopfield associative memory
- Pseudo-inverse memory networks

Contents:
- Mathematical derivations
- Python implementations
- Parameter studies
- Reproduced theoretical results
- Discussion and interpretation
